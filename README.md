# Square Runner
A puzzle game where the player takes control of a yellow smiley face, collects gold and bombs and gets to the exit. Beware of the the red smiley enemies!

## Controls
* WASD - Move Up, Left, Down and Right
* Space - Drop a bomb
* Escape - Reset
